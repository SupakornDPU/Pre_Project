ABOUT RECYCLE MACHINE

Reverse Vending Fun Machine

ตู้รับซื้อขยะรีไซเคิล อัตโนมัติ เปลี่ยนขยะรีไซเคิลเป็นเงินในรูปแบบต่างๆ โดยผู้ใช้สามารถนำขวด PET พลาสติกใส
หรือกระป๋องมาหยอดที่ตู้ เพื่อให้ เครื่องตรวจสอบคิดเป็นเงิน หรือสะสมเป็นแต้ม เพื่อนำไปแลกเป็น ความสนุกอื่นอีกต่อไป.

" วิธีการใช้งานตู้รีฟัน "

เริ่มต้นใช้งานง่ายๆ เพียงแค่ไม่กี่ขั้นตอน

กดที่ปุ่ม "REFUN" บนจอเพื่อเริ่มใช้งาน

กดเลือกเมนูสำหรับหยอดขวด ได้แก่สะสมแต้มสมาชิกและบริจาค

ใส่หมายเลขโทรศัพท์ เพื่อสะสมแต้มไว้แลกของรางวัล

สแกนบาร์โค้ด โดยการพลิกขวด หันด้านบาร์โค้ดขึ้นให้เห็นในจอ

กดปุ่ม "REFUN" อีกครั้ง เพื่อยืนยันเมื่อใส่ขวดครบ

เลือกรับหรือไม่รับใบเสร็จ ตามความต้องการ

Refun Machine : Reverse Vending Fun Machine

“เปลี่ยนการทิ้ง เป็นพลังสร้างสรรค์”

การรีไซเคิลที่ง่ายขึ้นในพื้นที่ที่คุ้นเคย สนุกมากขึ้นด้วยการ เปลี่ยนขยะเป็นเงินที่มีช่องทางให้เลือกรับได้อย่างทันสมัย ชัดเจน
ขึ้นด้วยการระบบตรวจสอบอัตโนมัติ การออกแบบตัวเครื่องและ การใช้งานที่ทำให้มุมมองของของที่จะทิ้งนั้นเปลี่ยนไป
